# mjbot.Plugins.Public

mjbot 的公共插件仓库。包含官方提供的开源插件示例与实现。

## 开发指南

如果你是第一次接触 mjbot 插件开发，请阅读 [开发指南 (DEVELOPMENT_GUIDE.md)](DEVELOPMENT_GUIDE.md)。

该指南包含：
- 环境准备
- 项目结构说明
- 如何创建第一个插件
- 指令系统与参数解析
- 配置文件管理
- 消息监听与多轮对话

## 快速开始

1. 克隆本仓库：
   ```bash
   git clone https://github.com/BJMANIA/mjbot.Plugins.Public.git
   ```
2. 打开 `mjbot.Plugins.Public.slnx` 或 `dependencies/MilkiBotFramework/MilkiBotFramework.sln` (视具体工作流而定)。
3. 参考 `src/mjbot.Plugins.Public/Plugins/DemoPlugin.cs` 进行开发。
